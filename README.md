# Thinky-AI

Thinky-AI is an AI-powered assistant and monitoring Discord bot designed to act as a lightweight chat companion while centralizing automated status reports from other services within the server ecosystem.

## Key Features

* **AI Chat Companion:** Direct interaction via dedicated channels with hot-swappable models (`!modelo fast` for quick replies and `!modelo care` for analytical tasks).
* **Automated Bot Monitoring:** Aggregates and reports automated events, updates, and status changes from other infrastructure bots (such as `GameServer-Bot` and system updates).
* **Lightweight & Efficient:** Optimized to run seamlessly on low-cost or legacy hardware (currently powered by Arch Linux).
